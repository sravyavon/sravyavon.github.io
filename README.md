## Getting Started

### Local Development

To run this website locally:

1. Clone the repository:
```bash
git clone https://github.com/sravyavon/sravyavon.github.io.git
cd sravyavon.github.io
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python 3
python3 -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have http-server installed)
http-server
```

3. Visit `http://localhost:8000` in your browser

## Customization

To customize your website:

1. Edit the HTML files to change content
2. Modify `styles.css` to change the appearance
3. Add new pages as needed
4. Update contact information in `contact.html`

## Deploying to GitHub Pages

Since this repository is named `sravyavon.github.io`, GitHub Pages automatically publishes the content from the main branch to your GitHub Pages site.

1. Make changes locally
2. Commit and push to GitHub:
```bash
git add .
git commit -m "Update website"
git push origin main
```

3. Visit your site at: `https://sravyavon.github.io`
